T-Pipes

A terminal-based data pipeline tool for the modern geek.

View on GitHub

Features

Modular Blocks

Chain sources, processors, and outputs to build powerful workflows.

Terminal Visualization

View structured data tables and raw output instantly in your shell.

Export Options

Seamlessly export your data to JSON, CSV, XML, and HTML.

Registry

Save and re-run your favorite pipelines with a simple command.

Usage

bash 
# Run a pipeline
python main.py example.yaml

# Register a pipeline
python main.py register example.yaml --name mypipe

# Run registered pipeline
python main.py run mypipe

DSL Example

pipeline.yaml 
- type: http_source
  config:
    url: https://api.example.com/data
- type: json_parser
- type: filter
  config:
    key: category
    value: tech
- type: print
